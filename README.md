# Gue

## Theory
```
Playground lives inside iframe
Playground runs in its own folder
Playground .vue files gets injected with data-gid="" on every element on project load
App parses vue files in project and links edition and hovering in iframe with ids
Changes are made in vue files on runtime
Before project / app close, remove data-gid="" from every file in project
```

### Other features
```
Iframe will be injected in runtime
Drag and drop elements
List of all possible html standard elements
CSS style edition
Live watch for code changes in project
Preview of running state
Own view for vue-router
Floating windows for edition
Export to electron app for win, mac, mas, cordova
```
